# TestFlow — MVP de Plataforma de Gestão de Testes de Software

Trabalho de conclusão da disciplina **Gestão Ágil de Projetos e Produtos**  
Pós-graduação — Vanessa Carvalho

---

## Sobre o Produto

O **TestFlow** é uma plataforma web de gestão de testes de software voltada para equipes de QA e desenvolvimento de pequenas e médias empresas. Centraliza casos de teste, execuções, rastreamento de defeitos e dashboards de qualidade em um único lugar.
